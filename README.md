
# Laboratório de Resolução de Problemas – UFF (25.2)

Este repositório contém as soluções desenvolvidas por mim durante os estudos da disciplina **Laboratório de Resolução de Problemas** da turma **25.2** do curso de Ciência da Computação da **Universidade Federal Fluminense (UFF)**.

Todos os exercícios foram resolvidos em **Python**, como 
parte do aprendizado de lógica de programação e resolução
de problemas computacionais.

O conteúdo aqui presente tem caráter **educacional** e reflete
meu processo de aprendizado ao longo da disciplina.
